README de `retrato.com/media/personas`

Esta carpeta contiene TODAS las fotos de perfiles.

Cada persona tiene su carpeta:
- `anahi`
- `colectivo`
- `hanna-s`
- `linda`
- `paola`
- `regina-o`
- `samuel`
- `ubaldo`

Dentro de cada carpeta usa estos nombres:
- `imagen_principal.jpg` (foto grande de la pagina)
- `retrato.jpg` (miniatura usada en el listado principal)
- `galeria_1.jpg`
- `galeria_2.jpg`

Para reemplazar imagenes sin romper nada:
1. Entra a la carpeta de la persona.
2. Sustituye archivos conservando exactamente el mismo nombre.
3. Recarga con `Ctrl + F5`.
